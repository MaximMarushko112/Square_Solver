void print_result ( int  result,
double  root1,
double  root2

Выводит результаты решения уравнения.

Если корней нет, выводит "NULL". Если корней 1 или 2, выводит их. Если корней бесконечно много, выводит "(-inf; +inf)".

Parameters
[in]resultКоличество корней уравнения
[in]root1,root2

int read_coef ( double *  n)

Получает значение одного коэфицента.

Если он введён корректно, передаёт его по указателю n и возвращает 0. В противном случае возвращает значение ошибки ввода.

Parameters
[out]n

◆ read_coefs()

int read_coefs ( struct SquareTrinomial *  eq)

Принимает все коэфиценты квадратного трёхчлена.

Сперва просит пользователя ввести коэфиценты. Если все коэфиценты введены правильно, передаёт их по указателю eq и возвращает 0. Иначе выводит сообщение "DURAK" и возвращает 1.

Parameters
[out]eq
